```
npm install
```

### Build

Build the web assets and sync with the Capacitor native projects:

```
npm run build
npx cap sync
```

### Run

Connect an iOS device via USB and run the app from IDE or Command Line Interface (CLI).


#### IDE
Open the native projects in corresponding IDEs (Xcode) and hit the "Run" button:

```
npx cap open ios
```

#### CLI
Or alternatively run the projects via CLI (iOS):

```
npx cap run ios
```
